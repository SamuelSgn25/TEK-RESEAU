# Tek-Reseau Website

Ce projet est le site web officiel de l'association estudiantine **Tek-Reseau**, dont le but est de démocratiser l'accès aux technologies réseau, à l'administration système et à la programmation réseau.

## 🚀 Fonctionnalités

- **Accueil Moderne** : Design premium avec glassmorphism, animations fluides et présentation de l'association.
- **Gestion des Membres** :
  - Importation de listes via fichier **PDF** (extraction automatique des noms).
  - Ajout manuel de nouveaux membres.
  - Gestion du Bureau Exécutif.
- **Suivi des Activités** :
  - Création et gestion de séances d'activités (TP, conférences, etc.).
  - Système de présence en temps réel avec statuts (Présent, Absent, Abandon).
  - Inscription rapide des membres aux séances.
- **Espace Administration** :
  - Authentification sécurisée par email/mot de passe.
  - Gestion du profil administrateur (changement de nom, email).
  - Dashboard interactif.

## 🛠️ Technologies Utilisées

- **React** (avec Vite)
- **Lucide React** (pour les icônes)
- **PDF.js** (pour l'extraction de données PDF)
- **Vanilla CSS** (pour un design personnalisé et performant)
- **Local Storage** (pour la persistance des données en local)

## 📦 Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/SamuelSgn25/TEK-RESEAU.git
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
